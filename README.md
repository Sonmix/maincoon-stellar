# 🌌 Maincoon Stellar

Веб-приложение для заводчиков мейн-кунов. Ведите карточки питомцев, прогнозируйте вязки, отслеживайте помёты и стройте родословные — всё в одном месте, прямо в браузере.

---

## Что умеет

### Питомник
- Карточки питомцев с фото, EMS-кодом окраса, генетической формулой
- Оценка экстерьера и характера
- Генетические тесты (HCM, PKD, SMA и другие)
- Родословная до 5 поколений (60 предков)
- Титулы и теги

### Прогноз вязки
- Расчёт вероятных окрасов потомства по генетике обоих родителей
- Квадрат Паннета для ключевых локусов
- Коэффициент инбридинга по формуле Райта (5 поколений)
- Анализ совместимости по генетическим тестам

### Журнал
- Календарь событий: течки, вязки, ожидаемые и фактические роды, заметки
- Автоматический расчёт даты ожидаемых родов (+63 дня от вязки)
- Фильтрация по питомцам и типам событий

### Пометы
- Учёт помётов с датой рождения, матерью и отцом
- Карточки котят: окрас, пол, вес, статус (ожидает / продан / остаётся)
- Перевод котёнка в питомник одной кнопкой

### Карта поколений
- Интерактивный SVG-граф родственных связей всего питомника
- Масштабирование и перемещение мышью
- Переход к карточке питомца по клику на узел

### Экспорт
- Выгрузка данных в форматах TXT, Markdown, HTML, ZIP

### Настройки
- Три темы: Stellar (звёздная), Dark, Light
- Интерфейс на русском и английском языках
- Сброс всех данных с тройным подтверждением

## Как запустить локально

Приложение — статика, для работы нужен локальный HTTP-сервер (не `file://` — из-за запросов к JSON-файлу).

**Python** (встроен в систему):
```bash
python -m http.server 8080
```
Затем открыть [http://localhost:8080](http://localhost:8080)

**Node.js**:
```bash
npx serve .
```

**VS Code**: установить расширение [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) и нажать «Go Live».

## Где хранятся данные

Все данные сохраняются в `localStorage` браузера — никакой регистрации, никаких серверов, ничего не уходит в интернет. Фотографии хранятся там же в формате base64.

> Мягкое ограничение localStorage — около 5 МБ. При большом количестве фотографий высокого качества лимит может быть достигнут.
